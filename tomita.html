CLIQUE Algorithm (Tomita et al.)

Description of the CLIQUE algorithm...

Pseudocode

        procedure CLIQUE(R, P, X)
            if P and X are both empty:
                report R as a maximal clique
            for each vertex v in P:
                CLIQUE(R ∪ {v}, P ∩ N(v), X ∩ N(v))
                P = P - {v}, X = X ∪ {v}
